## Requirements 
python 3.10.0

## Setting up your environment
MacOS 
```
$ brew install python3
```  
Linux
```
$ sudo apt-get update
$ sudo apt-get install python3.10
```
## Usage
```
$ python3 main.py
```

## Testing 
```
$ python3 unittests.py
```
or
```
$ curl localhost:8080/urlinfo/1/google.com
```
